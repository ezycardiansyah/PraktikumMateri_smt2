## ✨ Fitur

- ✅ **Responsive Design** - Bisa diakses dari HP, tablet, dan laptop
- ✅ **Preview Langsung** - Buka materi tanpa download (iframe embed)
- ✅ **Mobile Support** - Di HP otomatis buka di tab baru
- ✅ **Modal Popup** - Preview materi di desktop dengan tampilan modal
- ✅ **Hover Effect** - Animasi interaktif saat mouse diarahkan
- ✅ **14 Pertemuan** - Setiap mata kuliah memiliki 14 slot pertemuan
